Рішення складається з 2 проектів:
1) перший консольний - підтягує дані із файлу audio_video.json і пропонує користувачу меню, яке містить необхідні операції
2) другий проект - WinForms
- Upload - завантажує файл у json-форматі, json може містити 2 типи даних одночасно, audio_video.json - приклад коректного файла
- Remove - видаляє виділений рядок
- Clear - очищає дані
- Add - додає новий запис
- Edit - редагує вибраний запис
- Clear - очищає форму від даних
- Type - дає можливість вибрати тип даних
