This Console App provides some code snippets to deal with PDF files using
-iTextSharp.dll (class PdfEditor)
	Trim pages
	Convert Word to Pdf
	Copy Pdf from folder
-ILovePDF.dll (class PdfEditorILovePdf) (it installs JWT, Newtonsoft.Json, StringInterpolationBridge)
	Compress Pdf
-AsposePdf.dll (class PdfAspose)
	Parse Pdf to images

Testing folder provides code snippets which invoke methods from functionality classes with identified parameters
